# 5 Prompts Inteligentes — Implementación YAPÓ Dashboard

Copia y usa estos prompts para alimentar la IA de desarrollo o el equipo backend.

---

## Prompt 1: El Motor de Búsqueda Semántica

Configura el buscador de YAPÓ para que cuando un usuario escriba **"mi heladera hace escarcha"**, el sistema identifique el rubro **"Refrigeración"**, el nivel de urgencia **"Kavaju" (Rápido)**, y filtre automáticamente por profesionales activos en el barrio del usuario que tengan fotos/videos de reparaciones similares.

---

## Prompt 2: Sistema de Gestión Mbareté (Senso de Equipos)

Crea una lógica de administración para el perfil **"Mbareté"**. Permite que el líder vea una lista de su **"tropa"** (referidos), ordene por **Productividad** (servicios terminados) y **Calidad** (estrellas). Genera una **alerta** cuando un miembro baje de 3 estrellas para ofrecerle automáticamente una **"Beca de Capacitación"** de YAPÓ.

---

## Prompt 3: YAPÓ-METRIX (Métricas para Empresas)

Genera un módulo de estadísticas que cruce: **rubro_demanda vs barrio_oferta**. Si un usuario Enterprise busca **"Sajonia"**, el sistema debe devolver: *"Existen 45 albañiles, 0 Mbaretés líderes. Brecha detectada: Falta de coordinación de equipos. Recomendación: Iniciar programa de fidelización en la zona"*.

---

## Prompt 4: El Espacio de Comunidad y Sponsors

Diseña el **"YAPÓ Feed"**. Un muro donde usuarios suben problemas y profesionales responden. Implementa un sistema de **Posicionamiento Orgánico por Interés**: si el usuario activó su **Escudo Insurtech**, los primeros 3 comentarios en su feed deben ser de profesionales que aceptan ese seguro o sponsors relacionados a salud/seguridad.

---

## Prompt 5: El Flujo de Pago y Smart Contract

Define el protocolo de contratación: Al aceptar un presupuesto, YAPÓ genera un **link de pago**. Se **retiene el 50%**. El profesional sube una **foto del trabajo terminado** en la app. El cliente confirma. Se **libera el pago**. Si hay **disputa**, el sistema de **biometría y fotos** sirve como evidencia para el **arbitraje de YAPÓ**.
